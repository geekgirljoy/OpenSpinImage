Open Spin Image

Open Spin Image is a lightweight JavaScript "library" (it's really more a set of a few functions and a couple of CSS styles) that allows you to create a 360 degree spin image viewer from a set of images. Download from GitHub: https://github.com/geekgirljoy/OpenSpinImage

Example Spin Image

Press one of the arrows to rotate the image.

Another Example

These images can be 360 degree product scans or animation frames.
